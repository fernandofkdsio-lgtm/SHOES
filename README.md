# 🛍️ Product Image Selector (Frontend Project)

## 📌 Description

Ce projet est un composant front-end permettant d’afficher une chaussure avec plusieurs variantes visuelles.
L’utilisateur peut naviguer entre différentes images et changer l’apparence du produit de manière dynamique.

---

## 🚀 Fonctionnalités

* 🔁 Navigation entre les images (gauche / droite)
* 🎲 Affichage aléatoire des images au chargement
* 🎨 Sélection de variantes (ex : noir / blanc)
* ⚡ Mise à jour dynamique sans rechargement de page
* 📐 Mise en page avec Flexbox (`gap`, alignement, etc.)

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 🧠 Concepts abordés

* Manipulation du DOM (`querySelector`)
* Gestion des événements (`addEventListener`)
* Tableaux et logique d’index
* Randomisation (algorithme de mélange)
* Gestion des styles dynamiques (`backgroundImage`)
